# Image Attribution and Licensing

Use only images with explicit free licenses for commercial web use.

## Recommended source

- Wikimedia Commons: https://commons.wikimedia.org/

Search for Drammen images and verify each file's individual license page.

## Accepted licenses

- Public Domain
- CC0
- CC BY 4.0
- CC BY-SA 4.0

## Required attribution format

When license requires attribution, include:

- Title
- Author
- License name and link
- Source URL

Example:

- "Drammen River" by Jane Doe, CC BY-SA 4.0, via Wikimedia Commons.

## Project practice

- Store approved images in `static/img/drammen/`.
- Track each used image and attribution in this file.
- Do not use images without a clear license.
